How to Use

Works on ANY HTML element with data-scroll-section attribute.

HTML Structure

1. Add the SmoothScroll container:

<div data-smoothscroll data-smoothscroll-offset="80" data-smoothscroll-position="right"> </div>

2. Mark your sections:

<section data-scroll-section data-scroll-title="Home"> Content </section>

3. Load the scripts:

<script src="battersea-utils.js"></script> <script src="battersea-core.js"></script> <script src="battersea-tooltip.js"></script> <script src="battersea-smoothscroll.js"></script>
