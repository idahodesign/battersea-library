Technical Details

The component monitors header size changes intelligently.

üöÄ How It Works

  • On Init: Queries header element and measures initial height
  • On Scroll: Throttled check (100ms) detects size changes
  • On Resize: Debounced check (250ms) handles window resize
  • On Change: Reconnects IntersectionObserver with new rootMargin

‚ö° Performance

  • Uses throttle to limit scroll event checks
  • Only reconnects observer when size actually changes
  • 1px threshold prevents jitter from sub-pixel changes

All Options

Complete attribute reference.

üéõÔ∏è Available Attributes

  • data-smoothscroll-header-selector NEW
    CSS selector for header element (e.g., "#header", ".navbar")
  • data-smoothscroll-offset
    Static offset in pixels (ignored if header-selector is used)
  • data-smoothscroll-position
    "left" or "right" (default: "right")
  • data-smoothscroll-duration
    Animation duration in milliseconds (default: 800)
  • data-smoothscroll-easing
    "linear", "ease-in", "ease-out", "ease-in-out" (default: "ease-in-out")
  • data-smoothscroll-hide-mobile
    Hide on mobile screens (default: true)
