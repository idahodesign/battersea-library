The navigation menu is loaded from an external HTML file, making it easy to:

  • ‚úÖ Reuse the same menu across multiple pages
  • ‚úÖ Update the menu in one place
  • ‚úÖ Keep your page HTML clean and maintainable
  • ‚úÖ Switch between different menu structures easily

Available Menu Files:

  • nav-menu-simple.html - 2-level menu (Home, Products, Company, etc.)
  • nav-menu-full.html - 4-level menu with flyouts and accordions

üìù HTML Structure

<!-- Your page header -->
<header class="battersea-header" data-header>
  <div class="battersea-header__container">
    <div class="battersea-header__brand">
      <!-- Logo here -->
    </div>
    
    <!-- Navigation container (no data attributes) -->
    <nav class="battersea-header__nav" id="navContainer">
      <!-- Menu loaded here -->
      <!-- The loaded menu file includes data-horizontal-nav -->
    </nav>
  </div>
</header>

Note: The data-horizontal-nav attribute and configuration are in the menu files themselves, not on the container.

üîß JavaScript Loader

<script>
// Simple menu loader function
async function loadMenu(menuFile) {
  try {
    const response = await fetch(menuFile);
    const html = await response.text();
    document.getElementById('navContainer').innerHTML = html;
    
    // Re-initialize Battersea after loading menu
    if (window.Battersea) {
      Battersea.init();
    }
  } catch (error) {
    console.error('Failed to load menu:', error);
  }
}

// Load menu before Battersea initialization
loadMenu('nav-menu-full.html').then(() => {
  console.log('‚úÖ Menu loaded successfully!');
});
</script>
