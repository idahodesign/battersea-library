üîß How to Use

Set the layout using the data-layout attribute:

Right Align (Default)

<header class="battersea-header" data-header data-layout="right-align"> <!-- Logo on left, nav on right --> </header>

Center Align

<header class="battersea-header" data-header data-layout="center-align"> <!-- Logo and nav both centered, nav below --> </header>

Center Stacked (NEW)

<header class="battersea-header" data-header data-layout="center-stacked"> <!-- Logo above nav, both centered, logo stacked vertically --> </header>

üí° Layout Guidelines

When to Use Right Align:

  • Standard business websites
  • Applications and dashboards
  • Sites with many navigation items
  • When horizontal space is limited

When to Use Center Align:

  • Portfolio websites
  • Creative agency sites
  • Personal blogs
  • Sites with few navigation items

When to Use Center Stacked:

  • Brand-focused websites
  • Landing pages
  • Minimal designs
  • When logo needs maximum prominence
  • Sites where vertical space is abundant

üé® Styling Tips

All layouts respond to the same CSS variables and custom styles:

/* Customize header */ .battersea-header { background: #fff; box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1); } /* Customize logo */ .battersea-header__logo { height: 40px; /* Adjust logo size */ } /* Layout-specific adjustments */ .battersea-header--center-stacked .battersea-header__brand { padding: 30px 0 15px; /* More top padding for stacked */ } .battersea-header--center-stacked .logo-text { font-size: 32px; /* Larger text for stacked */ }

üì± Mobile Behavior

All layouts automatically adapt for mobile devices:

  • Right Align: Stacks on very small screens
  • Center Align: Maintains center alignment
  • Center Stacked: Maintains vertical stack (already mobile-optimized)
