Getting Started

Quick Installation

Include the CSS and JavaScript files in your project:

<!-- CSS -->
<link rel="stylesheet" href="path/to/battersea-library-min.css">

<!-- Core JavaScript (Required) -->
<script src="path/to/battersea-utils.js"></script>
<script src="path/to/battersea-core.js"></script>

<!-- Component Scripts (Load only what you need) -->
<script src="path/to/battersea-slider.js"></script>
<script src="path/to/battersea-accordion.js"></script>

Or clone from GitHub:

git clone https://github.com/idahodesign/battersea-library.git

Simple Usage Example

Components are initialized automatically using data attributes:

<!-- Accordion Example -->
<div data-accordion>
  <div class="battersea-accordion__item">
    <button data-accordion-trigger>Section 1</button>
    <div data-accordion-content>
      <p>Content goes here...</p>
    </div>
  </div>
</div>

<!-- Slider Example -->
<div data-slider data-slider-autoplay="true">
  <div class="battersea-slider__track">
    <div class="battersea-slider__slide">Slide 1</div>
    <div class="battersea-slider__slide">Slide 2</div>
  </div>
</div>

No JavaScript configuration required! Components auto-initialize on page load.

Theme Customization

Customize the library using CSS variables or LESS variables:

/* CSS Custom Properties */
:root {
  --battersea-primary: #667eea;
  --battersea-secondary: #764ba2;
  --battersea-text: #333;
}

/* Or use LESS variables */
@primary-color: #667eea;
@secondary-color: #764ba2;

/* Dynamic theming with less.js */
less.modifyVars({
  '@primary-color': '#ff6b6b',
  '@secondary-color': '#4ecdc4'
});
