Installation

Include the required files in your HTML:

<link rel="stylesheet" href="battersea-library.css">
<script src="battersea-utils.js"></script>
<script src="battersea-core.js"></script>
<script src="battersea-accordion.js"></script>
Usage

Add the accordion markup to your HTML:

<div data-accordion data-accordion-multiple="false">
	<div data-accordion-item>
		<div data-accordion-header>Header</div>
			<div data-accordion-content>
				<div>Content</div>
			</div>
		</div>
	</div>

HTML Structure

<div data-accordion data-accordion-multiple="false">
	<div data-accordion-item class="active">
		<div data-accordion-header>Section 1</div>
			<div data-accordion-content>
				<div>Content for section 1</div>
			</div>
		</div>
 
		<div data-accordion-item>
			<div data-accordion-header>Section 2</div>
			<div data-accordion-content>
				<div>Content for section 2</div>
			</div>
		</div>
	</div>

Attributes

Attribute Values Default Description
data-accordion - required Initialises the accordion component
data-accordion-multiple true / false false Allow multiple sections to be open simultaneously

Item Classes

  • active - Add to data-accordion-item to open by default

Events

The accordion dispatches a custom event when sections are toggled:

const accordion = document.querySelector('[data-accordion]');

accordion.addEventListener('battersea:accordionToggle', (e) => {
  console.log('Item:', e.detail.item);
  console.log('Is open:', e.detail.isOpen);
});

Customisation

Style the accordion using standard CSS:

/* Container */
[data-accordion-item] {
  background: white;
  border: 1px solid #e0e0e0;
  margin-bottom: 10px;
}

/* Header */
[data-accordion-header] {
  padding: 1rem;
  cursor: pointer;
  background: #f8f9fa;
}

/* Active header */
[data-accordion-item].active [data-accordion-header] {
  background: #007bff;
  color: white;
}

/* Content wrapper */
[data-accordion-content] {
  padding: 0 1rem;
}
