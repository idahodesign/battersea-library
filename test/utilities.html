Accessibility

Font size adjustment tool. Normally sits in the pre-header bar; shown here standalone to test its base styling.

The accessibility component provides a dropdown slider for adjusting the page font size. It stores the preference in localStorage.

Smooth Scroll

Navigation dots on the right side of the page. Each dot links to a section with data-scroll-section. The active section is highlighted as you scroll.

Configuration

<div data-smooth-scroll
  data-smoothscroll-position="right"
  data-smoothscroll-duration="800"
  data-smoothscroll-easing="ease-out"
  data-smoothscroll-hide-mobile="true">
</div>

The dot navigation on the right of this page is the live component. Each dot corresponds to a section heading.

Back to Top

Circular button that fades in as you scroll down. Clicking it smoothly scrolls back to the top of the page.

Configuration

<div data-back-to-top></div>

Scroll down the page to see the button appear in the bottom-right corner.

NavData

Core service that parses the navigation DOM into a JSON model. Used by Breadcrumbs and PageNav. Not a visible component -- it works behind the scenes.

Configuration

<!-- NavData auto-initialises when battersea-nav-data.js is loaded -->
<!-- It fires battersea:navDataReady when the data model is built -->

NavData provides sequential refs and breadcrumb paths for all pages in the navigation menu.
