Navigation Components

Header, menus, breadcrumbs and page navigation -- all running with default base styles.

Header

The header above is using the Header component with data-sticky and data-shrink enabled. Scroll down to see it shrink. The mobile menu toggle appears at smaller screen widths.

Configuration used

<header data-header
  data-sticky="true"
  data-shrink="true"
  data-shrink-offset="100"
  data-shrink-height="60"
  data-layout="right-align">

Breadcrumbs

Auto-generated from the page navigation. The breadcrumb trail at the top of this page is the live component.

Configuration

<div data-breadcrumbs></div>

The breadcrumb component reads the navigation structure via NavData and generates the trail automatically.

Page Navigation

Top level navigation

Jumps between top-level sections. Useful when you want to give people a way to move across major areas of the site.

Category page navigation

Stays within the current category. If you're in the Navigation section, it only moves between Navigation pages. Handy when you want people to explore related content.

All Pages

Prev/next navigation between pages. Shown below using sequential mode.
