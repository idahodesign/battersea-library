Accordion

Collapsible content sections. Single-open mode by default.

What does this component do?

The accordion creates collapsible sections that expand and collapse with smooth transitions. Only one section stays open at a time in single mode.

How is it configured?

Add data-accordion to the container. Use data-accordion-multiple="true" to allow multiple open sections. Add the active class to pre-open a section.

Does it support keyboard navigation?

Yes. Use Tab to move between headers and Enter or Space to toggle sections open and closed.

Tabs

Tabbed content panels. Click a tab to switch the visible content.

Tabs organise content into separate panels so users can switch between sections without leaving the page. Each tab maps to a content panel by ID.

Keyboard accessible, auto-initialised via data attributes, and works with dynamically loaded content. The active tab is highlighted automatically.

Add data-tabs to the container. Use data-tabs-nav for the tab buttons and data-tabs-content for the panels. Match data-tab and data-tab-content values.
