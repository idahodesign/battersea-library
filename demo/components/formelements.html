Form Elements Component

Custom form controls that look good and play well with FormValidation. Toggle switches, sliders, date and time pickers, and colour swatches — all vanilla JS, all keyboard accessible.

Live Demonstrations

Each element works standalone or inside a validated form

Toggle Switch

Wraps a standard checkbox in a styled switch. The checkbox stays in the DOM so forms submit as normal. Click the switch or use Space to toggle.

Range Slider

A custom slider that replaces the native range input. Drag the handle, click the track, or use arrow keys. The value snaps to the step interval.

Date Picker

Click the input to open a calendar. Navigate months with the arrows, click a day to select. Dates outside the min/max range are greyed out. Set data-datepicker-native="true" to use the browser's built-in date input instead.

Time Picker

A dropdown with scrollable hour and minute columns. The minute step controls the intervals shown — set it to 15 and you'll see 00, 15, 30, 45. Switch to 12-hour format with data-timepicker-format="12" to add an AM/PM column.

Colour Swatch

A grid of preset colours to pick from. Click the preview circle to open, click a swatch to select. When data-colourswatch-picker="true" is set, a "+" button lets you open a full colour picker with a hue bar, saturation/brightness area, and hex input.

FormValidation Integration

Wrap the elements in a data-form-validation form and add data-validate rules as normal

Configuration & Options

Toggle Switch

Place data-toggle on a <input type="checkbox">:

AttributeValuesDefaultDescription
data-toggle—requiredActivates toggle enhancement
data-toggle-onString"On"Label text when checked
data-toggle-offString"Off"Label text when unchecked
data-toggle-checked—absentIf present, starts checked

Range Slider

Place data-slider-input on a hidden input:

AttributeValuesDefaultDescription
data-slider-input—requiredActivates slider
data-slider-minNumber0Minimum value
data-slider-maxNumber100Maximum value
data-slider-stepNumber1Step increment
data-slider-valueNumberMidpointInitial value
data-slider-labelString"{value}"Display template. {value} is replaced with the number.

Date Picker

Place data-datepicker on a text input:

AttributeValuesDefaultDescription
data-datepicker—requiredActivates date picker
data-datepicker-formatDD/MM/YYYY, MM/DD/YYYY, YYYY-MM-DDDD/MM/YYYYDate display format
data-datepicker-minDate string—Earliest selectable date
data-datepicker-maxDate string—Latest selectable date
data-datepicker-nativetruefalseUse the browser's native date input

Time Picker

Place data-timepicker on a text input:

AttributeValuesDefaultDescription
data-timepicker—requiredActivates time picker
data-timepicker-format24 / 122424-hour or 12-hour (AM/PM) format
data-timepicker-stepNumber15Minute intervals
data-timepicker-minHH:MM—Earliest selectable time
data-timepicker-maxHH:MM—Latest selectable time

Colour Swatch

Place data-colourswatch on a hidden input:

AttributeValuesDefaultDescription
data-colourswatch—requiredActivates colour swatch
data-colourswatch-coloursComma-separated hex8 default coloursPreset colour options
data-colourswatch-pickertruefalseShow full colour picker via "+" button
data-colourswatch-valueHex colourFirst colourInitially selected colour

Custom Events

EventDetailDescription
battersea:formElementChange { name, value, type } Fired on the container when any element changes. type is "toggle", "slider", "datepicker", "timepicker", or "colourswatch".

Usage Examples

Toggle Switch
<div data-form-elements>
  <input type="checkbox" name="agree" data-toggle>

  <!-- Custom labels -->
  <input type="checkbox" name="mode"
    data-toggle
    data-toggle-on="Enabled"
    data-toggle-off="Disabled"
    data-toggle-checked>
</div>
Range Slider
<div data-form-elements>
  <input type="hidden" name="volume"
    data-slider-input
    data-slider-min="0"
    data-slider-max="100"
    data-slider-step="5"
    data-slider-value="50"
    data-slider-label="{value}%">
</div>
Date Picker
<div data-form-elements>
  <!-- Custom calendar -->
  <input type="text" name="date"
    data-datepicker
    data-datepicker-format="DD/MM/YYYY"
    data-datepicker-min="01/01/2026"
    data-datepicker-max="31/12/2026">

  <!-- Native browser input -->
  <input type="text" name="date"
    data-datepicker
    data-datepicker-native="true">
</div>
Time Picker
<div data-form-elements>
  <!-- 24-hour, 15-minute steps -->
  <input type="text" name="time" data-timepicker>

  <!-- 12-hour with AM/PM -->
  <input type="text" name="time"
    data-timepicker
    data-timepicker-format="12"
    data-timepicker-step="30">
</div>
Colour Swatch
<div data-form-elements>
  <!-- Preset swatches -->
  <input type="hidden" name="colour"
    data-colourswatch
    data-colourswatch-colours="#f00,#0f0,#00f,#ff0">

  <!-- With full colour picker -->
  <input type="hidden" name="colour"
    data-colourswatch
    data-colourswatch-picker="true"
    data-colourswatch-value="#667eea">
</div>
With FormValidation

Add both data-form-validation and data-form-elements to the form, then use data-validate as normal:

<form data-form-validation data-form-elements>
  <input type="text" name="date"
    data-datepicker
    data-validate="required">

  <input type="hidden" name="colour"
    data-colourswatch
    data-validate="required">

  <input type="checkbox" name="agree"
    data-toggle
    data-validate="required">

  <button type="submit">Submit</button>
</form>

Accessibility Features

Built with inclusivity in mind

⌨

ARIA Roles

Toggles use role="switch", sliders use role="slider", and swatch grids use role="radiogroup" with proper aria-checked states.

Tab

Focus Visible

Every interactive element shows a clear focus ring when navigated by keyboard. No ambiguity about where you are.

Esc

Escape to Close

All dropdowns (date picker, time picker, colour swatch) close with the Escape key. Focus returns to the trigger.

↔

Arrow Key Navigation

Sliders respond to arrow keys for precise adjustment. Colour swatch grids can be navigated with arrows.
