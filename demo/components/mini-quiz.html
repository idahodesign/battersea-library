Configuration

Container Attributes

Configure the quiz using data attributes on the container element:

Attribute Values Default Description
data-quiz - required Initialises the Mini Quiz component
data-quiz-data JSON or URL - Inline JSON array/object, or a URL to a .json file
data-quiz-csv URL - Path to a CSV file with quiz questions
data-quiz-title String - Quiz heading text
data-quiz-description String - Short intro text below the title
data-quiz-timer Number (seconds) 0 Countdown in seconds. 0 means no timer
data-quiz-hidden true / false false Hide quiz until a reveal button is clicked
data-quiz-results inline / summary / both both How results are displayed after submission
data-quiz-shuffle true / false false Randomise question order on load
data-quiz-shuffle-options true / false false Randomise option order within each question

Question Types

Each question uses a type field (in JSON/CSV) or data-quiz-type attribute (in HTML):

Type Renders As Answer Format
boolean Two large side-by-side buttons Single string (e.g. "True")
choice Radio buttons Single string (e.g. "Canberra")
multi Checkboxes with "select all that apply" Pipe-separated string or JSON array

Custom Events

The component fires these events on the container element:

Event Detail Description
battersea:quizReady { questionCount, hasTimer, isHidden } Fired after the quiz DOM is built
battersea:quizReveal { timestamp } Fired when a hidden quiz is revealed
battersea:quizAnswer { questionId, value, type } Fired when the user selects or changes an answer
battersea:quizSubmit { score, total, percentage } Fired when the quiz is submitted
battersea:quizTimerExpired {} Fired when the countdown reaches zero

Usage Examples

HTML Markup

Define questions directly in your markup:

<div data-quiz data-quiz-title="My Quiz">
  <div data-quiz-question
       data-quiz-type="choice"
       data-quiz-answer="Canberra">
    <p>Capital of Australia?</p>
    <div data-quiz-option>Sydney</div>
    <div data-quiz-option>Canberra</div>
    <div data-quiz-option>Melbourne</div>
  </div>
</div>
JSON File

Point to a JSON file containing your questions:

<div data-quiz
     data-quiz-data="path/to/questions.json"
     data-quiz-timer="120">
</div>

The JSON file structure:

{
  "title": "Quiz Title",
  "questions": [
    {
      "type": "choice",
      "question": "Your question?",
      "options": ["A", "B", "C"],
      "answer": "B",
      "explanation": "Optional explanation."
    }
  ]
}
CSV File

Load questions from a CSV file:

<div data-quiz
     data-quiz-csv="path/to/questions.csv"
     data-quiz-hidden="true"
     data-quiz-timer="90">
</div>

CSV format uses pipe-separated options and answers:

id,type,question,options,answer,explanation
q1,choice,"Which is biggest?","Cat|Dog|Horse",Horse,"Horses are large."
q2,multi,"Select mammals:","Dog|Snake|Cat","Dog|Cat","Dogs and cats."
Custom Styling with CSS Variables

Override the default look with CSS custom properties:

:root {
  --quiz-accent-color: #667eea;
  --quiz-correct-color: #27ae60;
  --quiz-incorrect-color: #e74c3c;
  --quiz-warning-color: #f39c12;
  --quiz-bg: #ffffff;
  --quiz-border-color: #e0e0e0;
  --quiz-border-radius: 8px;
  --quiz-option-bg: #f8f9fa;
  --quiz-option-hover-bg: #e9ecef;
  --quiz-option-selected-bg: #e8f0fe;
}
