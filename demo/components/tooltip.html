Tooltips on Different Elements

Tooltips aren't just for buttons — you can attach them to links, inline text, or any HTML element. The component handles the positioning automatically.

Here's a tooltip on a regular link within a paragraph of text. You can also put one on inline with your text, or on an inline span — the dashed underline is a nice visual cue that something is there.

Long Tooltip Text

Tooltips automatically wrap longer text. They're best kept short, but if you need a bit more detail, they handle it gracefully.

Key Features

⇱

Four Positions

Place tooltips on any side of the trigger — top, bottom, left, or right. Defaults to top if you don't specify.

~

Smooth Animation

Tooltips fade in and out with a gentle transition. No abrupt popping or jarring movements.

A11y

Focus Support

Tooltips appear on keyboard focus as well as hover, so keyboard and screen reader users get the same information.

➤

Auto-Initialisation

Add data-tooltip to any element and the tooltip is created automatically. No extra JavaScript needed.

★

Custom Styling

Apply a custom CSS class with data-tooltip-class to match your design system or create themed tooltips.

{ }

Zero Dependencies

Pure vanilla JavaScript. Lightweight, fast, and no external libraries to worry about.

Configuration & Options

Data Attributes

Configure tooltips using these data attributes on the trigger element:

Attribute Values Default Description
data-tooltip string required The text content shown inside the tooltip
data-tooltip-position top / bottom / left / right top Which side of the element the tooltip appears on
data-tooltip-class CSS class name - Adds a custom CSS class to the tooltip element for custom styling

Basic Usage

The simplest tooltip — just one attribute:

<button data-tooltip="Save your changes">Save</button>

With Position

Specify where the tooltip should appear:

<button data-tooltip="I appear below"
  data-tooltip-position="bottom">
  Hover me
</button>

With Custom Class

Apply your own styling with a custom class:

<button data-tooltip="I'm special"
  data-tooltip-position="top"
  data-tooltip-class="tooltip-warning">
  Warning
</button>

<style>
.tooltip-warning {
  background: #f44336;
  colour: white;
}
</style>

Usage Examples

Icon Button with Tooltip

Great for icon-only buttons where the user needs a label:

<button data-tooltip="Settings"
  data-tooltip-position="bottom"
  aria-label="Settings">
  &#9881;
</button>

<button data-tooltip="Delete"
  data-tooltip-position="bottom"
  aria-label="Delete">
  &#10005;
</button>

<button data-tooltip="Download"
  data-tooltip-position="bottom"
  aria-label="Download">
  &#8595;
</button>
Inline Help Text

Add contextual help to form labels or tricky terms:

<label for="email">
  Email Address
  <span data-tooltip="We'll never share your email with anyone"
    data-tooltip-position="right"
    style="cursor: help; border-bottom: 1px dashed #999;">
    (why do we need this?)
  </span>
</label>
<input type="email" id="email">
Navigation Link Tooltips

Give additional context to navigation items:

<nav>
  <a href="/dashboard"
    data-tooltip="View your personal dashboard"
    data-tooltip-position="bottom">
    Dashboard
  </a>
  <a href="/reports"
    data-tooltip="Generate and download reports"
    data-tooltip-position="bottom">
    Reports
  </a>
  <a href="/settings"
    data-tooltip="Manage your account preferences"
    data-tooltip-position="bottom">
    Settings
  </a>
</nav>
Tooltip on a Table Header

Explain column meanings without cluttering the table:

<table>
  <thead>
    <tr>
      <th data-tooltip="Full name of the customer"
        data-tooltip-position="top">Name</th>
      <th data-tooltip="Monthly recurring revenue in GBP"
        data-tooltip-position="top">MRR</th>
      <th data-tooltip="Customer lifetime value"
        data-tooltip-position="top">LTV</th>
    </tr>
  </thead>
  <tbody>
    <!-- Table data -->
  </tbody>
</table>
