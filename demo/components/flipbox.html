Key Features

3D

CSS 3D Transforms

True 3D perspective flipping using CSS transforms. Smooth, hardware-accelerated, and works at 60fps.

↔

Two Directions

Flip horizontally (left-to-right) or vertically (top-to-bottom). Set with a single data attribute.

☛

Hover or Click

Choose the trigger that fits your design. Hover for desktop-focused layouts, click for touch-friendly interfaces.

{ }

Custom Events

Dispatches battersea:flipboxFlip with flip state, so you can react to flips in your own code.

Configuration & Options

Data Attributes

AttributeValuesDefaultDescription
data-flipbox-requiredInitialises the flipbox on the container
data-flipbox-directionhorizontal / verticalhorizontalThe axis the card flips on
data-flipbox-triggerhover / clickhoverWhat triggers the flip animation

Required Structure

<div data-flipbox data-flipbox-direction="horizontal" data-flipbox-trigger="hover">
  <div data-flipbox-front>Front content</div>
  <div data-flipbox-back>Back content</div>
</div>
