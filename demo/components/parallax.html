Parallax Component

Smooth background parallax scrolling that adds depth and motion to your pages

Standard Parallax (Speed 0.5)

The default speed setting — a good balance between movement and subtlety. The background image scrolls at half the speed of the page content.

Standard Parallax

This background moves at half the scroll speed, creating a natural sense of depth. It's the most commonly used setting and works well for hero sections and content breaks.

Slow Parallax (Speed 0.2)

A much subtler effect — the background barely moves as you scroll. Nice for situations where you want just a hint of depth without too much drama.

Slow Parallax

With a speed of 0.2, the background only shifts a small amount. You'll notice it if you're looking for it, but it won't distract from the content. Great for long-form reading pages.

Fast Parallax (Speed 0.8)

A more pronounced effect — the background moves at nearly the same speed as the scroll. Use this sparingly, as it can feel quite intense.

Fast Parallax

At 0.8 speed, the background moves quite dramatically. This works well for short, punchy sections where you want to grab attention — but you probably wouldn't want a full page of this.

Key Features

~

Smooth Scrolling

Uses requestAnimationFrame for buttery-smooth background movement with no janky stuttering or tearing.

⚙

Adjustable Speed

Set the parallax speed from 0 (no movement) to 1 (full scroll speed) using a simple data attribute. Fine-tune it to your liking.

➤

Auto-Initialisation

Just add data-parallax to any element with a background image and you're done. No JavaScript configuration needed.

{ }

Zero Dependencies

Built with pure vanilla JavaScript. No jQuery, no external libraries, no bloat. Just clean, efficient code.

★

GPU Accelerated

Uses CSS transforms for hardware-accelerated rendering, keeping your animations smooth even on less powerful devices.

↻

Mobile Aware

The parallax effect degrades gracefully on mobile devices where performance can be a concern. No jarring behaviour on touch screens.

Configuration & Options

Data Attributes

Configure the parallax component using these data attributes:

Attribute Values Default Description
data-parallax - required Initialises the parallax effect on the element
data-parallax-speed 0 to 1 0.5 Controls how fast the background scrolls relative to the page. 0 means no movement, 1 means it scrolls at the same speed as the content.

Basic Usage

Add a parallax background to any element — just make sure it has a background image and a height set:

<div data-parallax data-parallax-speed="0.5"
  style="background-image: url('your-image.jpg');
         background-size: cover;
         background-position: center;
         min-height: 400px;">
  <!-- Your content here -->
</div>

Tips

  • Always set background-size: cover so the image fills the container
  • Use background-position: center to keep the focal point centred
  • Give the element a min-height so it's tall enough to show the effect
  • Lower speed values (0.1–0.3) work best for subtle, professional layouts
  • Higher speed values (0.7–0.9) create dramatic effects — use them sparingly

Usage Examples

Hero Section with Parallax

A common pattern — a full-width hero banner with a parallax background:

<section data-parallax data-parallax-speed="0.4"
  style="background-image: url('hero-bg.jpg');
         background-size: cover;
         background-position: center;
         min-height: 600px;
         display: flex;
         align-items: center;
         justify-content: center;">
  <div style="text-align: center; color: white;">
    <h1>Welcome to Our Site</h1>
    <p>Something compelling goes here</p>
  </div>
</section>
Content Break Parallax

Use parallax sections to break up long pages of content:

<!-- Regular content section -->
<section class="content">
  <p>Your content here...</p>
</section>

<!-- Parallax break -->
<div data-parallax data-parallax-speed="0.3"
  style="background-image: url('break-bg.jpg');
         background-size: cover;
         min-height: 300px;">
</div>

<!-- More content -->
<section class="content">
  <p>More content here...</p>
</section>
Parallax with Overlay

Add a semi-transparent overlay for better text readability:

<div data-parallax data-parallax-speed="0.5"
  style="background-image: url('bg.jpg');
         background-size: cover;
         min-height: 400px;
         position: relative;
         display: flex;
         align-items: center;
         justify-content: center;">
  <div style="background: rgba(0,0,0,0.5);
              padding: 2rem;
              border-radius: 12px;
              color: white;
              text-align: center;">
    <h2>Your Heading</h2>
    <p>Content is easy to read thanks to the overlay</p>
  </div>
</div>
Multiple Parallax Sections

Each parallax element is independent — you can have several on the same page with different speeds:

<div data-parallax data-parallax-speed="0.3"
  style="background-image: url('bg-1.jpg'); ...">
  <!-- Slow section -->
</div>

<div class="content">
  <p>Regular content between parallax sections</p>
</div>

<div data-parallax data-parallax-speed="0.7"
  style="background-image: url('bg-2.jpg'); ...">
  <!-- Faster section -->
</div>
