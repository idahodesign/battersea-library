Timeline Component

Vertical and horizontal timelines with scroll-reveal animations, date badges, and custom markers

Live Demonstrations

The timeline in action across different layouts

Vertical Alternating (Company History)

Items zigzag left and right of a centre line. Each card can have a date badge, icon marker, image, and description. On mobile, everything stacks into a single column.

Key Features

↕

Dual Layout

Vertical or horizontal, set with a single data attribute. The horizontal layout scrolls with navigation arrows.

⇄

Alternating

Vertical timelines can alternate items left and right of the centre line. Collapses to single column on mobile.

◉

Scroll Reveal

Vertical items animate in via the Animation component. Horizontal items reveal with a staggered entrance, then snap-scroll with active highlighting.

★

Custom Markers

Each point on the line can show an emoji, number, or text. Leave it blank for a simple dot.

📷

Media Support

Cards can include images alongside titles and descriptions. Dates are optional too.

⚙

Themeable

Full CSS custom property control over colours, sizes, spacing, and shadows. Match any design system.

Configuration & Options

Container Attributes

Set these on the main timeline wrapper element:

Attribute Values Default Description
data-timeline - required Initialises the timeline component
data-timeline-direction vertical / horizontal vertical Layout direction. Horizontal falls back to vertical on mobile
data-timeline-alternate true / false true Alternate items left/right on vertical desktop layouts
data-timeline-animation-offset Integer (px) 100 Scroll offset before items animate (horizontal only; vertical uses the Animation component)

Item Attributes

Set these on each timeline entry:

Attribute Element Description
data-timeline-item Item wrapper Marks an element as a timeline entry (required)
data-timeline-date Item wrapper Date or label text shown as a badge. Omit for undated items
data-timeline-icon Item wrapper Custom content for the marker on the line (emoji, number, text). Omit for a plain dot
data-timeline-intro Item wrapper Short intro text shown on horizontal cards. Full content appears in the panel below when active
data-timeline-image <img> Marks an image inside the item for card display

Usage Examples

Basic Vertical Timeline

The simplest setup. Add data-timeline to a container and data-timeline-item to each entry:

<div data-timeline>
  <div data-timeline-item data-timeline-date="2024">
    <h3>Event Title</h3>
    <p>Description of what happened.</p>
  </div>

  <div data-timeline-item data-timeline-date="2025">
    <h3>Another Event</h3>
    <p>More details here.</p>
  </div>
</div>
Horizontal Process Flow

Set data-timeline-direction="horizontal" for a snap-to-centre layout with a content panel below. Use data-timeline-intro for short text on the card:

<div data-timeline data-timeline-direction="horizontal">
  <div data-timeline-item data-timeline-date="Step 1" data-timeline-intro="Short intro">
    <h3>Research</h3>
    <p>Full description shown in the panel below when active.</p>
  </div>

  <div data-timeline-item data-timeline-date="Step 2" data-timeline-intro="Another intro">
    <h3>Design</h3>
    <p>Detailed content for the panel.</p>
  </div>
</div>
Items with Images

Add data-timeline-image to any <img> inside an item:

<div data-timeline-item data-timeline-date="2023">
  <img src="photo.jpg" alt="Description" data-timeline-image>
  <h3>Event with Photo</h3>
  <p>The image appears at the top of the card.</p>
</div>
Custom Styling with CSS Variables

Override the defaults to match your design:

:root {
  --battersea-timeline-line-color: #e0e0e0;
  --battersea-timeline-marker-bg: #ff6b6b;
  --battersea-timeline-card-bg: #f8f9fa;
  --battersea-timeline-card-shadow: 0 4px 20px rgba(0, 0, 0, 0.12);
  --battersea-timeline-date-bg: #ff6b6b;
  --battersea-timeline-card-radius: 12px;
  --battersea-timeline-gap: 60px;
}
Listening to Events

The component dispatches custom events you can listen for:

const timeline = document.querySelector('[data-timeline]');

// Fires when the component is ready
timeline.addEventListener('battersea:timelineReady', (e) => {
  console.log('Direction:', e.detail.direction);
  console.log('Items:', e.detail.itemCount);
});

// Fires when a horizontal item becomes active (centred)
timeline.addEventListener('battersea:timelineItemActive', (e) => {
  console.log('Active item', e.detail.index);
});

// Fires when horizontal arrows are clicked
timeline.addEventListener('battersea:timelineScroll', (e) => {
  console.log('Scrolled:', e.detail.direction);
});
