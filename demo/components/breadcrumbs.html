Breadcrumbs Component

Auto-generated breadcrumb trail from your site's navigation menu

Live Demonstrations

The breadcrumb at the top of this page is a working example

How It Works

The breadcrumb trail you see above the page title is generated automatically. It reads the site's navigation menu, works out where you are, and builds the trail for you. No manual configuration needed on each page.

For this page the trail reads: Home | Navigation | Breadcrumbs because the Breadcrumbs component sits inside the Navigation category in the main menu.

CSS Variables

Configuration & Options

Data Attributes

Configure the breadcrumbs component using these data attributes:

Attribute Values Default Description
data-breadcrumb - required Initialises the breadcrumb component on the element
data-breadcrumb-separator Any string | The character displayed between breadcrumb items

CSS Custom Properties

Override these variables to customise the appearance:

Variable Default Description
--battersea-breadcrumb-text #666 Colour of the current page text
--battersea-breadcrumb-link #667eea Colour of breadcrumb links
--battersea-breadcrumb-link-hover #4c5fd5 Hover colour of breadcrumb links
--battersea-breadcrumb-separator-color #999 Colour of the separator character
--battersea-breadcrumb-font-size 0.875rem Font size of the breadcrumb text
--battersea-breadcrumb-spacing 0.5rem Gap between items and separators
--battersea-breadcrumb-padding 0.75rem 0 Padding around the breadcrumb trail

Dependencies

The Breadcrumbs component requires the NavData service, which reads the site's navigation menu. Make sure battersea-nav-data.js is loaded before battersea-breadcrumbs.js.

Usage Examples

Basic Breadcrumb

Drop this element anywhere on the page. It auto-detects the current page and builds the trail:

<div data-breadcrumb></div>
Custom Separator

Change the separator character with the data-breadcrumb-separator attribute:

<!-- Forward slash -->
<div data-breadcrumb data-breadcrumb-separator="/"></div>

<!-- Chevron -->
<div data-breadcrumb data-breadcrumb-separator="&#8250;"></div>

<!-- Arrow -->
<div data-breadcrumb data-breadcrumb-separator="&#8594;"></div>
Custom Styling

Override the CSS custom properties to match your design:

:root {
  --battersea-breadcrumb-link: #3498db;
  --battersea-breadcrumb-link-hover: #2980b9;
  --battersea-breadcrumb-text: #95a5a6;
  --battersea-breadcrumb-separator-color: #bdc3c7;
  --battersea-breadcrumb-font-size: 0.8rem;
}
Required Scripts

Include the core files, NavData service, and the breadcrumbs script:

<script src="battersea-utils.js"></script>
<script src="battersea-core.js"></script>
<script src="battersea-nav-data.js"></script>
<script src="battersea-breadcrumbs.js"></script>

The NavData service must load before the Breadcrumbs component. It parses the navigation menu and makes the data available for the breadcrumb trail.

Accessibility Features

Built with inclusivity in mind

nav

Landmark Navigation

Uses <nav> with aria-label="Breadcrumb" so screen readers can identify it as navigation.

ol

Ordered List

The trail is an <ol> element, conveying the hierarchical order to assistive technology.

➤

Current Page

The last item uses aria-current="page" to identify the current location in the trail.

[ ]

Focus Indicators

Visible focus outlines on all breadcrumb links for keyboard navigation.
