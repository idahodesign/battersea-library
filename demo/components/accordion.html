Single Open Mode (Default)

Only one section can be open at a time. Opening a new section automatically closes the previous one.

What is the Accordion component?

The Accordion component allows you to create collapsible content sections that expand and collapse smoothly. Perfect for FAQs, content organisation, and space-saving layouts.

It features:

  • Smooth max-height transitions
  • Keyboard accessibility
  • Custom events for integration
  • No snapping or jerky animations
How do I install it?

Installation is straightforward. Include the required files in your HTML:

<link rel="stylesheet" href="battersea-library.css">
	<script src="battersea-utils.js"></script>
	<script src="battersea-core.js"></script>
	<script src="battersea-accordion.js"></script>

Then add the accordion markup to your page and it will initialise automatically!

Is it accessible?

Yes! The accordion is fully keyboard accessible and follows accessibility best practices:

  • Keyboard navigation with Tab and Enter
  • Clear visual focus indicators
  • Semantic HTML structure
  • Works with screen readers
  • WCAG 2.1 compliant

Multiple Open Mode

Allow multiple sections to be open simultaneously. Perfect for complex content where users may need to reference multiple sections.

Step 1: Installation

First, include the Battersea Library files in your project:

<!-- CSS -->
<link rel="stylesheet" href="path/to/battersea-library.css">

<!-- Core JavaScript (Required) -->
<script src="path/to/battersea-utils.js"></script>
<script src="path/to/battersea-core.js"></script>

<!-- Accordion Component -->
<script src="path/to/battersea-accordion.js"></script>
Step 2: HTML Markup

Add the accordion structure to your HTML:

<div data-accordion data-accordion-multiple="true">
  <div data-accordion-item>
    <div data-accordion-header>Section Title</div>
    <div data-accordion-content>
      <div>Section content goes here</div>
    </div>
  </div>
</div>
Step 3: Customisation

Customise the appearance with CSS:

[data-accordion-item] {
  margin-bottom: 10px;
  border-radius: 8px;
}

[data-accordion-header] {
  padding: 1rem;
  cursor: pointer;
  background: #f8f9fa;
  transition: background 0.3s;
}

[data-accordion-header]:hover {
  background: #e9ecef;
}

Configuration & Options

Data Attributes

Configure the accordion component using these data attributes:

Attribute Values Default Description
data-accordion - required Initialises the accordion component on the container element
data-accordion-multiple true / false false Allow multiple sections to be open simultaneously

Item Structure

Each accordion item must follow this structure:

<div data-accordion-item>
  <div data-accordion-header>Section Title</div>
  <div data-accordion-content>
    <div>
      Content wrapper required for proper height calculation
    </div>
  </div>
</div>

CSS Classes

The component automatically manages these classes:

Class Element Description
active data-accordion-item Applied when the section is expanded. Add manually to pre-open a section.

Custom Styling

Customise the appearance with your own CSS:

/* Container styles */
[data-accordion-item] {
  background: white;
  border: 1px solid #e0e0e0;
  margin-bottom: 10px;
  border-radius: 8px;
  overflow: hidden;
}

/* Header styles */
[data-accordion-header] {
  padding: 1rem 1.5rem;
  cursor: pointer;
  background: #f8f9fa;
  font-weight: 600;
  transition: background 0.3s;
}

[data-accordion-header]:hover {
  background: #e9ecef;
}

/* Active header */
[data-accordion-item].active [data-accordion-header] {
  background: #667eea;
  color: white;
}

/* Content wrapper */
[data-accordion-content] {
  padding: 0 1.5rem;
}

/* Inner content div */
[data-accordion-content] > div {
  padding: 1rem 0;
}
Listening to Events

Listen for toggle events to integrate with other functionality:

const accordion = document.querySelector('[data-accordion]');

accordion.addEventListener('battersea:accordionToggle', (e) => {
  console.log('Item toggled:', e.detail.item);
  console.log('Is now open:', e.detail.isOpen);
  
  // Custom logic here
  if (e.detail.isOpen) {
    // Section was opened
    console.log('Section expanded');
  } else {
    // Section was closed
    console.log('Section collapsed');
  }
});
