Key Features

☰

Multiple Items

Show several items at once — unlike a regular slider that shows one at a time. Perfect for product cards, team members, or portfolios.

⚙

Responsive Breakpoints

Set different item counts for desktop, tablet, and mobile. The carousel recalculates automatically when the window resizes.

▶

Autoplay

Optional auto-scrolling with a configurable interval. Pauses on hover so users aren't fighting with the carousel while browsing.

↔

Configurable Gap

Control the spacing between items with the data-multislider-gap attribute. Set it in pixels — the layout handles the rest.

☞

Touch Support

Swipe left and right on mobile to scroll through items. The interaction feels natural and responsive on touch screens.

{ }

Zero Dependencies

Pure vanilla JavaScript with no external libraries. Lightweight and fast, with no bloat to slow your page down.

Configuration & Options

Data Attributes

Configure the multi-slider using these data attributes on the container element:

Attribute Values Default Description
data-multislider - required Initialises the multi-slider component on the container
data-multislider-items number 3 Number of items visible on desktop
data-multislider-items-md number 2 Number of items visible on tablet (768px–1023px)
data-multislider-items-sm number 1 Number of items visible on mobile (< 768px)
data-multislider-gap pixels 20 Gap between items in pixels
data-multislider-autoplay true / false false Whether the carousel scrolls automatically
data-multislider-interval milliseconds 3000 Time between auto-scrolls (only used when autoplay is on)

Element Structure

The multi-slider uses a track and item system with arrow buttons:

Attribute Element Description
data-multislider-track div The scrollable track containing all items
data-multislider-item div Each carousel item (direct children of the track)
data-multislider-prev button Navigate to previous items
data-multislider-next button Navigate to next items

Basic Markup

Here's the minimum markup for a multi-slider:

<div data-multislider data-multislider-items="3"
  data-multislider-gap="20">
  <div data-multislider-track>
    <div data-multislider-item>Card 1</div>
    <div data-multislider-item>Card 2</div>
    <div data-multislider-item>Card 3</div>
    <div data-multislider-item>Card 4</div>
    <div data-multislider-item>Card 5</div>
  </div>
  <button data-multislider-prev>&#8249;</button>
  <button data-multislider-next>&#8250;</button>
</div>

Listening to Scroll Events

Hook into the multi-slider's navigation events:

const carousel = document.querySelector('[data-multislider]');

carousel.addEventListener('battersea:multisliderScroll', (e) => {
  console.log('Scrolled to index:', e.detail.currentIndex);
  console.log('Visible items:', e.detail.visibleItems);
  console.log('Total items:', e.detail.totalItems);
});
