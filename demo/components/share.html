Configuration & Options

Data Attributes

Configure the share component using these data attributes:

Attribute Values Default Description
data-share - required Initialises the share component on the container element
data-share-url URL string Current page URL The URL to share
data-share-title Text Page title Title text included in the share
data-share-text Text Same as title Description text for platforms that support it
data-share-platforms Comma-separated list All platforms Which platforms to show: x, facebook, linkedin, bluesky, whatsapp, email
data-share-style icons / pills icons Display style -- circular icons or labelled pills
data-share-direction horizontal / vertical horizontal Layout direction of the buttons

Events

Event Detail Description
battersea:shareClick { platform, url } Fired when a share button is clicked

Basic Structure

<div data-share
    data-share-url="https://example.com"
    data-share-title="Page Title"
    data-share-text="Description text"
    data-share-platforms="x,facebook,linkedin,bluesky,whatsapp,email"
    data-share-style="icons"
    data-share-direction="horizontal">
</div>

Usage Examples

Basic Share Buttons

The simplest setup -- add a single data attribute and get all six platforms:

<div data-share></div>

The component picks up the current page URL and title automatically.

Pill Style with Custom Platforms

Show labelled buttons for specific platforms:

<div data-share
    data-share-style="pills"
    data-share-platforms="x,linkedin,email"
    data-share-title="My Article"
    data-share-text="Worth reading.">
</div>
Vertical Sidebar

Stack buttons vertically for a sidebar or floating panel:

<div data-share
    data-share-direction="vertical"
    data-share-platforms="x,facebook,linkedin">
</div>
Listening for Share Events

Track which platforms get clicked:

document.querySelector('[data-share]')
  .addEventListener('battersea:shareClick', function(e) {
    console.log('Shared on:', e.detail.platform);
    console.log('URL shared:', e.detail.url);
  });
Include Files

Add these files to your page:

<!-- CSS -->
<link rel="stylesheet" href="battersea-library-min.css">

<!-- Core JavaScript (Required) -->
<script src="battersea-utils.js"></script>
<script src="battersea-core.js"></script>

<!-- Share Component -->
<script src="battersea-share.js"></script>
