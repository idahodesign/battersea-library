Key Features

⇅

Column Sorting

Click any header to cycle through ascending, descending and unsorted. Supports text, number and date types.

⚲

Global Filter

Type to search across every column at once. Results update as you type with a debounced input for smooth performance.

…

Pagination

Set a page size and the table splits into pages with navigation controls, page numbers and a row count summary.

↔

Column Resizing

Drag column borders to adjust widths. Uses fixed table layout with a minimum width to keep things readable.

☑

Row Selection

Checkboxes on each row with a select-all toggle. Selection state is tracked and available through custom events.

CSV

CSV Export

Export the current filtered and sorted view as a CSV file. One click download with properly escaped values.

Configuration & Options

Data Attributes

Configure the DataTable component using these data attributes on the container element:

Attribute Values Default Description
data-table - required Initialises the DataTable component
data-table-sortable true / false true Enable column sorting
data-table-filterable true / false false Show the global search input
data-table-page-size Number 0 Rows per page. Set to 0 for no pagination
data-table-selectable true / false false Show row selection checkboxes
data-table-resizable true / false false Enable column resize by dragging
data-table-exportable true / false false Show the CSV export button
data-table-striped true / false true Alternating row background colours
data-table-csv URL - Path to a CSV file. The first row is used as column headers
data-table-data JSON array or URL - Inline JSON array, or a URL to a JSON file
data-table-columns JSON array auto-detect Column definitions for JSON mode

Column Attributes

These attributes go on individual <th> elements:

Attribute Values Default Description
data-column-key String col0, col1... Unique key for the column (used in JSON mode)
data-column-type string / number / date string Data type used for sorting
data-column-sortable true / false true Whether this specific column can be sorted

Custom Events

The component fires these events on the container element:

Event Detail Description
battersea:tableSort { column, direction } Fired when a column is sorted
battersea:tableFilter { query, results } Fired when the filter query changes
battersea:tablePageChange { page, pageSize } Fired when the page changes
battersea:rowSelect { selectedRows, row } Fired when row selection changes
battersea:tableExport { data, format } Fired when data is exported

Usage Examples

Basic HTML Table Enhancement

Write a normal HTML table and the component does the rest:

<div data-table data-table-sortable="true">
  <table>
    <thead>
      <tr>
        <th>Name</th>
        <th data-column-type="number">Age</th>
        <th>City</th>
      </tr>
    </thead>
    <tbody>
      <tr><td>Alice</td><td>30</td><td>London</td></tr>
      <tr><td>Bob</td><td>25</td><td>Bristol</td></tr>
    </tbody>
  </table>
</div>
JSON Data Source

Pass data as JSON attributes. Column definitions are optional; if omitted, they are auto-detected from the first row:

<div data-table
  data-table-sortable="true"
  data-table-filterable="true"
  data-table-columns='[
    {"key":"name","label":"Name"},
    {"key":"age","label":"Age","type":"number"}
  ]'
  data-table-data='[
    {"name":"Alice","age":30},
    {"name":"Bob","age":25}
  ]'>
</div>
JSON File Source

Point data-table-data at a URL instead of inline JSON:

<div data-table
  data-table-data="data/inventory.json"
  data-table-sortable="true"
  data-table-filterable="true"
  data-table-page-size="10">
</div>
CSV File Source

Point the component at a CSV file. The first row is treated as column headers:

<div data-table
  data-table-csv="data/team.csv"
  data-table-sortable="true"
  data-table-filterable="true"
  data-table-page-size="10">
</div>
Full-Featured Configuration

Enable all features at once:

<div data-table
  data-table-sortable="true"
  data-table-filterable="true"
  data-table-page-size="10"
  data-table-selectable="true"
  data-table-resizable="true"
  data-table-exportable="true"
  data-table-striped="true">
  <table>
    <thead>...</thead>
    <tbody>...</tbody>
  </table>
</div>
Listening to Events

Hook into table interactions with custom events:

var table = document.querySelector('[data-table]');

table.addEventListener('battersea:tableSort', function(e) {
  console.log('Sorted by:', e.detail.column);
  console.log('Direction:', e.detail.direction);
});

table.addEventListener('battersea:rowSelect', function(e) {
  console.log('Selected rows:', e.detail.selectedRows);
});

table.addEventListener('battersea:tableFilter', function(e) {
  console.log('Query:', e.detail.query);
  console.log('Matches:', e.detail.results);
});

  --datatable-accent-color: #e94560;
}

Accessibility Features

Built with inclusivity in mind

Tab

Keyboard Navigation

Tab to sortable headers, Enter or Space to toggle sort direction. Tab through pagination and filter controls.

aria

ARIA Sort

Sortable headers use aria-sort to announce the current sort state: none, ascending or descending.

SR

Live Announcements

Sort changes and filter results are announced to screen readers via an aria-live region.

[ ]

Focus Indicators

Clear focus outlines on all interactive elements for keyboard and switch-device users.
